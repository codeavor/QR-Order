App that lets you order just by scanning a QR Code.

## Setup

- Clone the project.
- Run: `npm nstall` to install the dependencies.
- Run: `npm start` to start the app.
- To run the test, run: `npm test`.
- To create a test coverage, run: `npm run coverage`
